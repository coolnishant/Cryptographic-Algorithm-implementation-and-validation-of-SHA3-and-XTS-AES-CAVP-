Prerequisite:
	* Windows/Linux platform
	* JDK 8 or more
Tools required: 
	* NetBeans 8.2 or more

Steps to run the file
a) Extract the zip file
b)	1) Open the project in NetBeans	
	2) Open TestReadFile.java
	3) Change Primary File location to the test vectors.
	4) Run the file.
	5) Test Vector output file is generated at the 	primary location set at step 3).
	6) Verify the result
		i) Browse to Primary Location.
		ii) Open outputted file: which is in format of inputfilenameout.rsp
		iii) Check ok count gives number of successful outputs.
